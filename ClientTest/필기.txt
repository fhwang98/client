웹페이지(웹문서)를 제작하는 언어

1. HTML
	- 웹문서 작성 언어
	- 페이지의 골격을 작성
	- 페이지의 출력 내용을 작성
	
2. CSS
	- 디자인 언어
	- HTML 페이지 + 서식 작성 역할

3. JavaScript
	- 프로그래밍 언어(자바같은애)
	- 웹페이지 + 프로그래밍 기능



HTML, Hyper Text Markup Language
- 하이퍼 텍스트 마크업 언어
- 프로그래밍 언어(X) > 프레젠테이션 언어(O) : 무언가를 표현하는 기능을 가진 언어
- Hyper Text > 링크
- Markup > 언어 표기법: <키워드>
		- HTML, XHTML, XML, VoiceVML, **ML ... 
- SGML > 메타 언어 > 다른 언어를 정의할 수 있는 기능을 가진 언어
		- HTML, XML (SGML을 기반으로 된 대중화가 된 언어는 이 두가지..)
		- HTML > 표현에 집중
		- XML > 데이터 관리에 집중


HTML 주관 단체
- W3C, World Wide Web Consortium, WWW 모든 기술 생성 + 주관
- https://w3.org
- WHATWG(왓웍) > 최근 표준

- 팀 버너스리

- HTML 1.0 ~ HTML 5(2014년)
- HTML 4.01 > 가장 대중화된 버전(90년대 말)
- XHTML 1.0 > HTML + XML > 문법 강력하게 준수
- HTML 5
- 현재 사용하는 HTML은 버전이 없음
- HTML Living Standard > 버전 명시(X) > 기능 조금씩 추가 발전
- 현재 사용하는 HTML은 무슨 버전인가요? > 그냥 HTML
